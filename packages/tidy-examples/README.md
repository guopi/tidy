# `tidy-examples`

> examples of tidyjs

## Usage

### start example server
```bash
$ yarn ts ./src/get-start.ts
```
run `yarn ts <ts-file-path>` to start example  
   
   
### test server
use httpie/curl to test server response
eg:
```bash
   $ http post :3000/test/123 'Cookie:name1=value1;cookie2=value2'
```
